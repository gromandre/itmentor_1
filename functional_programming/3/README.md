# Weather CLI

CLI-утилита для получения почасового прогноза погоды по городам.

## Возможности
- Принимает список городов через аргументы командной строки
- Поддерживает загрузку городов из файла
- Подготавливает данные для дальнейшей обработки (геокодер, прогноз, CSV)

## Требования
- Python 3.10+

## Запуск

### Ввод городов через файл
Файл должен содержать **один город на строку**:

```text
Москва
Тула
Рязань
```

Команды запуска:
```bash
python weather.py -f cities.txt
python weather.py --file cities.txt
```

### Ввод городов через аргументы CLI

```bash
python weather.py Москва Тула "Рязань"
```

### Комбинированный вариант

```bash
python weather.py Москва --file cities.txt
```

## Формат ввода
- Пустые строки игнорируются



## Примечание
Этот README описывает только ввод данных. Логика запроса погоды, кэширования и сохранения CSV реализуется отдельно.
